# VERUM Landing Page

Public-facing promotional website. Provides:
- Project overview and explainability demos (Grad-CAM examples).
- Privacy/GDPR section (no payload storage, SHA-256 hashes only).
- Deep link CTA → `tg://resolve?domain=VerumBot` to open Telegram directly.

## Stack
React + Vite + Tailwind CSS (recommended) or plain HTML/Tailwind for simplicity.

## Deployment
Vercel / Netlify / GitHub Pages — connect repo for automatic CI/CD on push to `main`.
